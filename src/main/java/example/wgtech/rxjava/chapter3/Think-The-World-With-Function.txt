클로저, 루비 등 함수형 프로그래밍 언어를 활용해 기존 프로그래머들이 세상을 함수로 생각할 수 있는 다섯가지 추상적인 단계를 파악해보자.

1. 왜
2. 전환
3. 양도하라
4. 열심히보다는 현명하게
5. 진화하라

위 다섯가지 단계는 작은 문제부터 차근히 해결하다보면 더 깔끔하고 추상화가 잘 된 코드를 작성할 수 있다는 것을 뜻한다.
몇 개의 핵심적인 개념을 이해하고 반복적으로 연습하면, 함수형 프로그래밍에 좀 더 익숙해질 것이다.

Python 언어는 함수형 프로그래밍 개념을 지원하고 있다는 것을 참고하여, 개념 파악에 익숙해질 필요가 있다.
